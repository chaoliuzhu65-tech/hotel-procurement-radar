# Hotel Procurement Price Radar - 酒店采购比价助手

> **开源酒店采购比价AI助手** - 让经验定价、虚假询比价成为历史

---

## 一句话介绍

基于 OpenClaw + 飞书生态的酒店采购成本管控工具，通过AI自动查询农产品批发价、电商零售价，帮助财务/采购/厨房快速核价、询比价。

---

## 核心功能

| 功能 | 说明 |
|------|------|
| 🌾 农产品批发价 | 农业农村部、新发地、惠农网等9个数据源 |
| 🛒 电商价格对比 | 淘宝、京东、拼多多实时价格 |
| 📊 多维表格导出 | 一键追加到飞书多维表格 |
| 🔄 自动容灾 | 主源失败自动切换备用数据源 |
| 🏨 酒店全品类 | 蔬菜/肉类/水产/调料/工程材料/酒店用品 |

---

## 快速开始

### 方式1：直接使用（推荐）

在飞书机器人对话框直接说：

```
"查价格"
"查一下黄瓜价格"
"上传报价单"
"批量查蔬菜50种"
```

### 方式2：部署到自己的OpenClaw

**Step 1**: 安装到 OpenClaw Skills 目录

```bash
# 克隆到本地 skills 目录
cd ~/.openclaw/workspace/skills
git clone https://github.com/delonix-ai/hotel-procurement-radar.git
```

**Step 2**: 配置飞书多维表格

创建飞书多维表格，参考下方「数据表结构」章节

**Step 3**: 开始使用

直接对机器人说「查xx价格」即可

## 架构说明

```
用户（任意设备）
     ↓ 自然语言
OpenClaw 云端引擎
     ↓
Web搜索 → 农业农村部/新发地/电商平台
     ↓
智能路由 → 自动选择可用数据源
     ↓
兜底机制 → 主源失败自动切换
     ↓
飞书多维表格（结果存储）
```

## 数据表结构

在飞书多维表格中创建以下字段：

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 商品名称 | 文本 | - |
| 规格 | 文本 | 如：一级/山东 |
| 品类 | 单选 | 蔬菜/肉类/水产/调料等 |
| 批发价(元/斤) | 数字 | - |
| 淘宝价(元/斤) | 数字 | - |
| 京东价(元/斤) | 数字 | - |
| 拼多多价(元/斤) | 数字 | - |
| 建议采购价(元/斤) | 文本 | 如：1.2-1.5 |
| 风险提示 | 文本 | - |
| 查询时间 | 日期 | - |
| 数据来源 | 文本 | - |
| 查询用途 | 多选 | 核价/询价/比价/成本分析 |

---

## 容灾机制

当主数据源不可用时，自动切换到备用：

```
农业农村部 → 新发地 → 商务预报
新发地 → 农业农村部 → 惠农网
淘宝 → 京东 → 拼多多
京东 → 淘宝 → 拼多多
```
