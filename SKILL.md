```

                      ▼
```

**特点**：
- ✅ **无需安装** 任何软件
- ✅ **任意设备** 均可使用（手机/电脑/平板）
- ✅ **任意城市** 均可使用（无城市限制）
- ✅ **自动容灾** 主源失败自动切换
- ⚠️ 零售价参考，非批发价

**特点**：

- ❌ **需要安装**（本地电脑）
- ❌ **仅限Windows/Mac**

### 3.2 容灾机制（自动切换）

```python
# 伪代码：智能路由 + 自动容灾
async def query_price(product, region=None):
    """
    智能查询：自动选择数据源 + 容灾切换
    """
    # Step 1: 确定品类路由
    category = identify_category(product)  # 农产品/电商/工程/酒店用品
    
    # Step 2: 获取该品类数据源列表
    sources = DATA_SOURCES[category]
    
    # Step 3: 按优先级尝试
    for source in sources:
        try:
            # 尝试查询
            result = await query(source, product, region)
            if result.is_valid():
                return result
        except SourceUnavailable:
            # 自动切换到fallback
            continue
    
    # Step 4: 全都失败，返回友好提示
    return PriceResult.unavailable(
        message=f"所有{product}数据源暂时不可用，请稍后再试"
    )
```
