## 五、多维表格容错设计

### 5.1 自动检测 + 自动创建

```python
async def ensure_bitable():
    """
    多维表格容错处理：
    1. 检查是否已配置bitable_token
    2. 如果没有，自动创建新的多维表格
    3. 如果有但无权限，提示用户授权
    4. 如果表格不存在，自动重建
    """
    bitable_token = config.get("bitable_token")
    
    if not bitable_token:
        # 自动创建新表格
        bitable = await create_new_bitable(name="酒店采购比价记录")
        config.set("bitable_token", bitable.app_token)
        return bitable
    
    try:
        # 验证表格是否可访问
        table = await bitable.get(app_token)
        return table
    except PermissionDenied:
        # 无权限，提示用户
        return None, "需要您授权多维表格权限"
    except TableNotFound:
        # 表格被删除，自动重建
        return await create_new_bitable(name="酒店采购比价记录")
```
