# 🏨 Hotel Procurement Price Radar - 酒店采购比价助手

> **开源酒店采购比价AI助手** - 让经验定价、虚假询比价成为历史

---

## 一句话介绍

基于 OpenClaw + 飞书生态的酒店采购成本管控工具，通过AI自动查询农产品批发价、电商零售价，帮助财务/采购/厨房快速核价、询比价。

---

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🌾 农产品批发价 | 农业农村部，新发地、惠农网等9个数据源 |
| 🛒 电商价格对比 | 淘宝、京东、拼多多实时价格 |
| 📸 图片搜索 | 发送商品图片，识别同款比价 |
| 📊 多维表格导出 | **自动创建**，无需手动配置 |
| 🔄 自动容灾 | 主源失败自动切换备用数据源 |
| 🏨 酒店全品类 | 蔬菜/肉类/水产/调料/工程材料/酒店用品 |

---

## 🚀 快速开始

### 方式1：直接使用（推荐）

在飞书机器人对话框直接说：

```
"查价格"

"上传报价单"
"批量查蔬菜50种"
```


## 🏗️ 架构

```
用户（任意城市/任意设备）
     ↓ 自然语言
OpenClaw 云端引擎
     ↓
Web搜索 → 农业农村部/新发地/电商平台
     ↓
智能路由 → 自动选择可用数据源
     ↓
自动容灾 → 主源失败自动切换
     ↓
飞书多维表格（自动创建，无需配置）
```

## 🔄 容灾机制

当主数据源不可用时，自动切换到备用：

```
农业农村部 → 新发地 → 商务预报
淘宝 → 京东 → 拼多多
新发地 → 农业农村部 → 惠农网
```

---

## 📋 多维表格自动容错

**无需手动配置！** SKILL会自动：

1. 检测是否已配置多维表格token
2. 如果没有，自动创建新的多维表格
3. 如果有但无权限，提示用户授权
4. 如果表格不存在，自动重建
