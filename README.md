# 🏨 Hotel Procurement Price Radar

> **开源酒店采购比价AI助手** - 让经验定价、虚假询比价成为历史

## 一句话

基于 OpenClaw + 飞书生态的酒店采购成本管控工具，通过AI自动查询农产品批发价、电商零售价，帮助财务/采购/厨房快速核价、询比价。

## ✨ 核心功能

| 功能 | 说明 |
|------|------|
| 🌾 农产品批发价 | 农业农村部，新发地、惠农网等9个数据源 |
| 🛒 电商价格对比 | 淘宝、京东、拼多多实时价格 |
| 📊 多维表格导出 | 一键追加到飞书多维表格 |
| 🔄 自动容灾 | 主源失败自动切换备用数据源 |
| 🏨 酒店全品类 | 蔬菜/肉类/水产/调料/工程材料/酒店用品 |

## 🚀 快速开始

### 直接使用（推荐）

在飞书机器人对话框直接说：

```
"查价格"
"查一下黄瓜价格"
"上传报价单"
```

### 部署到自己的OpenClaw

```bash
cd ~/.openclaw/workspace/skills
git clone https://github.com/chaoliuzhu65-tech/hotel-procurement-radar.git
```

## 📡 数据源（19个）

### 农产品（9个）
- ✅ 农业农村部（官方批发价）
- ✅ 北京新发地（批发市场）
- ✅ 商务预报（全国均价）
- ✅ 寿光蔬菜指数
- ✅ 猪价网
- 🔜 惠农网、一亩田

### 电商（4个）
- ⏳ 淘宝、天猫
- ⏳ 京东
- ✅ 拼多多
- ⏳ 1688
